# Web-Blocking - 网站屏蔽工具  
🚫 屏蔽你不想看到的网站

![项目截图](https://github.com/QsSama-W/Web-blocking/blob/main/%E6%95%88%E6%9E%9C%E5%9B%BE.png)  


## 🌟 核心功能  
1. **灵活过滤规则**  
   - 支持通配符匹配：`*.web.com/*`  
   - 精准路径屏蔽：`https://example.com/path`  
2. **规则管理系统**  
   ✔️ 支持规则文件的导入/导出  
   ✔️ 可视化规则列表编辑  


## 📖 使用说明  
1. **添加规则**  
   - 点击扩展图标 → 打开设置面板  
   - 输入网址或路径 → 点击添加
   - 支持批量导入JSON规则文件  


## 🚀 浏览器支持
- ✔️ Chromium内核浏览器
- ❌ Firefox
- ❌ Safari
